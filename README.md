# Data Literacy cho DPO tương lai ☕

Khóa học tự học (web tĩnh, **một file HTML**) giúp người làm **business / non-tech** đỡ ngợp về **data literacy**, từ đó tự tin làm tốt vai trò **Privacy / Data Protection Officer (DPO)** — bám chuẩn **DAMA-DMBOK** + pháp luật bảo vệ dữ liệu cá nhân Việt Nam (PDPL / NĐ 13/2023) và bối cảnh ngân hàng.

🔗 **Học ngay:** https://dpo101.tamduongs.com

## Nội dung
- **5 module · 35 bài · 5 bài trắc nghiệm** (guard-rail: đậu ≥7/10 mới mở module sau) · glossary Việt–Anh · template **ROPA / DPIA / Data Inventory** tải được.
- Lộ trình: Nền tảng dữ liệu (DAMA) → Hiểu IT (server, **SQL**, cloud, **ITGC/ITAC**, **ISO 27001**) → Privacy thực chiến (data mapping, **ROPA, DPIA, DSAR**, sự cố lộ lọt, luật VN) → BI/AI → **Vai trò DPO & ma trận RACI**.

## Đặc điểm kỹ thuật
- **1 file HTML tĩnh duy nhất** — không backend, không gọi mạng để chạy; link/QR ngoài chỉ kích hoạt khi người dùng bấm/quét.
- Layout dạng **trình chiếu (slide)**, animation, lưu **tiến độ trong localStorage** (theo trình duyệt).
- Deploy: web tĩnh trên **Vercel** (xem `vercel.json`). `/` rewrite tới `data-literacy.html`.
